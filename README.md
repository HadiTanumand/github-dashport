# برنامه ی احراز هویت با استفاده از اکانت گیتهاب
توضیحات نحوه ی دانلود و نصب به زبان انگلیسی در پایین تر نوشته شده است،
این برنامه با استفاده از کامپوننت های آماده ی متریال یوآی طراحی شده 
در مجموع هفت کامپوننت کوچک هست که بزرگترین آنها 90 خط می باشد و به راحتی دوباره قابل استفاده می باشند.
برای بک اند و دریافت توکن از 
Supabase 
استفاده شده که نحوه طراحی و کدزنی را بسیار راحت کرده است و سرعت خیلی بالایی هم دارد.
پس از اجرای برنامه در پورت سه هزار با صفحه ی لندگینگ و دکمه ی ورود مواجه میشوید جهت احراز هویت از طریق گیت هاب روی این دکمه کلیک کنید درصورت طولانی شدن و یا به مشکل خوردن کافی است صفحه را رفرش کنید
پس از احراز هویت دوباره به نرم افزار برمیگردید و بعد از چند ثانیه اکانت گیتهاب خودرا مشاهده میکنید،
میتوانید با کلیک روی دکمه ی صفحه ی پروفایل همه ی ریپازیتوری های خودرا ببینید و هر ریپازیتوری به صورت کارت می باشد میتوانید با زدن دکمه فایل زیپ را دانلود کنید.
برای خروج از حساب کاربری کافی است دکمه ی لاگ اوت را بزنید در غیر اینصورت چون یک کوکی ذخیره گشته هربار وارد نرم افزار بشوید به صورت خودکار شما را به اکانت خودتان متصل میکند.

ویژگی جدید:
همچنین شما می توانید نام ریپازیتوری مدنظر خودرا سرچ کنید و پیدا کنید در بین ریپازیتوری ها، این فیچر برای کسانی که تعداد ریپازیتوری های زیادی دارند کاربردی می باشد.
